# MSQ IT Website

A modern, responsive website for MSQ IT services built with Next.js and Tailwind CSS.

## Features

- Modern UI with light sky blue color scheme and network background imagery
- Responsive design for all device sizes
- Animated components and transitions
- Reusable React components for consistent styling
- SEO-friendly structure
- AI-powered chatbot with NinjaOne ticketing system integration

## Pages

- Home
- Services
  - Managed Services
  - ERP Consulting
- How We Work
- Contact
- About

## Technology Stack

- Next.js
- React
- Tailwind CSS
- JavaScript
- Azure OpenAI (for chatbot)
- NinjaOne API (for ticketing system)
